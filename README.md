### To build make sure:
-> python3 installed<br>
-> you have a C++23 compatible compiler<br>
-> you have cmake >= 3.30 installed<br>
-> you have vcpkg configured with cmake before build<br>
<br>
Than run:<br>
On Windows:<br>
python run.py<br>
<br>
On Linux, FreeBSD, macOS:<br>
python run.py<br>
<br>
if configured properly cmake should automatically<br>
trigger vcpkg to run when it should<br>
